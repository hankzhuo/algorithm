## 数组

数组类算法技巧：
- 暴力解法：双循环，不过时间复杂度比较高，通常为 O(n^2)，不推荐
- 空间换时间技巧：用空间（数组、变量、哈希表等）换时间的思路，减少时间复杂度
- 双指针技巧：一个快指针、一个慢指针，或者一个左边指针、一个右边指针等代表坐标
- 二分查找
- 二维数组、多维数组：比如 `[[],[]]` 
- 位运算
